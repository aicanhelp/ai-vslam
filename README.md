# ai-vslam

### setup environment
- For Mac OS
```
    cd scripts
    ./setup-macos.sh
```
- For Ubuntu
```
    cd scripts
    ./setup-ubuntu.sh
```


### Build openvslam
```
    cd scripts
    ./build-openvslam.sh
```

### Run exmaples
- Tracking and Mapping
```
    cd scripts
    ./run-examples.sh tracking_mapping
```
